## Repo structure
- `index.qmd` = homepage
- `proposal.qmd` = proposal
- `results.qmd` = results + figures
- `scripts/` = analysis scripts
- `data/` = small data only + data notes
- `figures/` = exported plots/images
- `docs/` = built website output (auto)

## How to collaborate (GitHub Lite)
1. Pull latest changes
2. Make edits
3. Commit with a clear message
4. Push

**Commit message examples**
- "Add dataset description to proposal"
- "Cleaned variable names in script 01"
- "Add Figure 2: correlation heatmap"
